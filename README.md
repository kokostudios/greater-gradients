# Greater Gradients
Greater Gradients is a gradient pack for Aseprite.

This pack has a total of 6 gradients, all characterized by their own mascots.
![Characters](https://img.itch.zone/aW1nLzI1ODUyMzQwLnBuZw==/original/DUCZsK.png)


\* Characters are an example and a representation of the gradients, they are not in the zip file.
# Installation instructions
Installing Greater Gradients is actually really easy, follow the instructions and you will have all of the gradients.
## Requirements
- [Aseprite](https://aseprite.org/) (Paid and Free, depending on how you got it)
## Installing Greater Gradients
- Download the Greater Gradients pack from this page
- Extract the files and set the extraction path to "%AppData%\Aseprite\extensions". (On some apps, you have to check the checkbox titled "create filename folder under selected folder" (or anything similar)

If you do this correctly, you will see the gradients in the dithering tab.
